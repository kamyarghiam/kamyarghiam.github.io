Projects

Syne:
Sign Language to Speech Converter

Collaborators: Saransh Chopra, Rhea Jain, Anish Krishnan
This project was developed in less than 24 hours as part of the annual Carnegie Mellon hackathon "HackCMU."

Our project converts sign language to speech, using machine learning and infrared detection. The program gathers (x,y,z) coordinates of 15 joints in the hand using a leap motion. We map these joints to 3D vectors relative to the palm. The vectors are normalized to ensure that hand size does not affect the accuracy of the system. Our neural network takes in the 45 data points and uses three dense hidden layers to categorize the hand gestures as one of 26 letters. The model was trained with thousands of readings for each letter and has over 99% accuracy.

Expert Engine:
Matching Projects to Experts

Collaborators: Mark Kong, Jonah Hendler, Zach Mines, Josh Kuperman

Expert Engine is an AI recommender system created for internal use at Experfy to help match experts to proposed projects. The program sifts through thousands of expert bios to determine whose skills are related to the projects that are bid to the company. Using Natural Language Processing and word2vec, the program cleans the input information, determines the skills necessary for the projects, and ranks the experts in order of who can meet and exceed those requirements.


Optimal Room Lighting

Collaborator: Josh Kuperman

In this project, we created an evolutionary program that finds the best lighting in a room for the cheapest cost. We took many characteristics into account, including brightness, location, light spread, cost, number of lights, obstacles, etc. The algorithm can be adjusted for mutation rate, population size per generation, number of generations, and obstacle locations. Our results showed that, on the room scheme shown in the video, three lights around the center with medium illumination provides optimal (and cheap) lighting.

Deciduo

Deciduo is a program that uses psychological inputs to learn and detect what foods/exercises make people happy and prevent them from quitting their diets. Using machine learning and Lagrange optimization, the program learns what types of foods/exercises maximize people's happiness over time while keeping them healthy. You can also use other people's algorithms to see what they would prefer.

Sentiment Analysis of President Trump's Tweets

Collaborator: Saransh Chopra

We extracted and analyzed over 37,000 tweets from president Trump. We wanted to compare the sentiments between the daytime and nighttime tweets. We used a Google API to calculate sentiment and found that the difference between the two groups was not statistically significant. As a note, we considered 6:00 PM - 6:00 AM to be nighttime.

Raspberry Pi/Arduino Facial Recognition

Collaborator: Sam Kaplan

In this project, we used facial recognition to light up LED buulbs. The main modules and languages in our project were Python, OpenCV, Serial, and the Arduino programming language. The Raspberry Pi sent serial commands to an Arduino to light up the blue LED bulb when a face was recognized, the green LED bulb when eyes were recognized, and the yellow one when there was more than one face.
